# 测试执行指南

## 测试流程

### 阶段1：测试原始版本 SKILL.md

```bash
# 确认当前是原始版本
cat SKILL.md | head -20
# 应该看到 description 中没有 "asking ONE question at a time"
```

1. 打开新的 Claude Code 会话
2. 执行：
   ```
   Use the business-spec-to-golden skill to process eval/testcases/case-03-simple.md
   ```
3. 用 `eval/testcases/case-03-answers.md` 中的预设答案回答问题
4. 完成后，将对话内容复制到 `eval/results/v1-original/session-log.md`

### 阶段2：切换到修改版本

```bash
# 应用修改版本
cp SKILL.md.new SKILL.md
```

1. 打开新的 Claude Code 会话
2. 执行相同的测试用例
3. 用相同的预设答案回答问题
4. 完成后，将对话内容复制到 `eval/results/v2-modified/session-log.md`
